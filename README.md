🎬 Film Rental Store Data Analysis (MySQL Project)
📌 Project Overview

This project focuses on analyzing a Film Rental Store database using MySQL to understand customer behavior, rental trends, and business performance.

The goal of this project is to perform SQL-based data analysis to generate insights that can help the business improve marketing strategies, customer retention, and revenue generation.

Through structured queries and relational database analysis, the project answers important business questions such as:

How customers behave when renting movies

Which films generate the most revenue

Which locations perform better

How marketing efforts impact rentals

Customer engagement and activity trends

🗂️ Database Schema

The project uses a relational database representing a film rental business.

Main tables include:

film – movie details

actor – actor information

category – movie genres

customer – customer information

rental – rental transactions

payment – payment records

inventory – store inventory

store – store locations

staff – store employees

These tables are connected using primary and foreign keys, enabling complex relational queries.
